Windows :

Scaricare l'ultima versione disponibile da :

https://sourceforge.net/projects/luabinaries/files/

Nella cartella Tools Executables troveremo il file .zip per Windows 32/64 bit

una volta scaricato bisognerà scompattarla in una cartella e impostare il percorso
nella variabile d'ambiente Path.
Una volta volta fatto ciò si consiglia di rinominare lua55.exe e luac55.exe
in lua.exe e luac.exe, cosi da poter utilizzare direttamente il comando lua.
Per eseguire senza compilare : lua Hello.lua
per compilare :  luac -o Hello Hello.lua
per eseguire dopo aver compilato : lua hello

MacOS :

https://www.lua.org/download.html

scarica l'ultima versione, scompatta l'archivio e segui le istruzioni :

cd lua-5.5.0

make all test

vai in lua-5.5.0->src

sposta i file luac e lua nella cartella -> /usr/local/bin/
